# CSE2138_Project_1
A program which takes hexadecimal bytes from a .txt file and turns them into specified data types with specified size and byte ordering.

./<executable> <input.txt> <endian_type> <data_type> <data_size>

<endian_type> : "l" , "b"
<data_type> : "u" , "i" , "fp"
<data_size> : "1" , "2", "3", "4"
